- lsb_release -a : 系统版本
- uname -a : 内核版本
- df -TH : 查看cpu资源和内存资源(转换显示方式)
- cd : 进入
- ls -la : 查看文件夹内荣(权限)(全部文件)
- mkdir : 创建目录
- touch : 创建文件
- vi : 编辑文件（包含创建文件）
- cat : 查看文件
- echo : 追加内容(>>)覆盖内容(>)
- rm -rf : 删除文件(目录)(强制)
- tar zcxvf : 压缩处理(通过gzip指令处理)(压缩)(解压缩)(显示过程)(使用归档名字)
- ps -ef | grep ' ' : 查看进程(关键字)
- kill -9 : 杀死进程(彻底)
